# Используются
 - cтатический HTML, SCSS
 - слайдер
 - сборщик Webpack

# Запуск 
На локальном сервере, например, через Live Server

На порту 3000:
 - npm run install - установка зависимостей
 - npm run build - сборка проекта
 - npm run dev - режим разработки
 - npm run start - запуск
